# Change Log

All notable changes to the "DB Viewer" extension will be documented in this file.

## [1.0.0] - 2025-11-08

### Added
- Initial release of DB Viewer
- Custom editor for SQLite database files (.db, .sqlite, .sqlite3)
- Visual table listing in sidebar
- Interactive data grid viewer with scrolling
- Table schema information display
- Row and column count statistics
- VS Code theme integration (light/dark mode support)
- Read-only mode for safe database viewing
- Performance optimization for large databases (1000 row limit)
- Pure JavaScript implementation using sql.js (no external dependencies)
- Support for tables with special characters and SQL keywords

### Features
- Click any .db file to open in the viewer
- Browse all tables via sidebar navigation
- View data in clean, formatted tables
- Sticky column headers for easy reference
- Automatic theme adaptation
