# Change Log

All notable changes to the "DB Viewer" extension will be documented in this file.

## [1.0.0] - 2025-11-08

### Added

- Initial release of DB Viewer
- Custom editor for SQLite database files (.db, .sqlite, .sqlite3)
- Visual table listing in the sidebar and a responsive data viewer
- Schema information and row/column counts
- Read-only viewing mode and VS Code theme integration
- Performance optimizations for large tables (first 1000 rows shown)
- Implemented using a pure JavaScript SQLite engine (`sql.js`)

## [Unreleased]

### Planned
